# Epic: 泛活动生态场景自定制建排器微流引擎

## 概述
活动不再是以特定写定发文系统来，这是一个可装配各类表单签到投票等的自立型活体生命系统服务引擎。不仅实现发起控制更具备闭环扫码场源调配管理等一揽全套。

## 故事列表

### US-011: DIY图示搭流发起复合复杂规则大活动的管理创建引擎流端工具
**As a** 校事后勤运营与学外校友召接项目负责带头管理人等各种活动设计管理决策和统属协调方等干事操盘组等操作角们等在规划校庆节点大筹等各种重要群等场建任务发起方。

**I want** 从微服务表引擎内快速选择并挂制各种模块例如需要收集意见调查卷再加入入场人员名次人数拦截器和抽签功能及配选图票大擂功能以及组车与定物筹集资源登记表的拼搭拼合生成一种全新复合复杂逻辑闭合型的一整链路大功能应用包页面向整个群内推送公映和邀通，以免自己请外源程序现做导致工期不达标的情况再发阻等。

**Acceptance Criteria:**
1. **Given** 身在一个新的全功能搭配型自定义微流程创建管理系统控制大台。并在白板中间托入组合如打卡收签+报项表提交并加挂审核阀段模块然后存稿发布发布后。
   **When** 进入终极发布模式确认发放到对应群体小组织面中可见列区划并生效等之后，
   **Then** 小程序的校友终端刷新即见新设栏区并如其按其配流被顺利规整入栏指点通过不遇乱跳错误代码崩坏卡阻停步和丢失数据。

### US-012: 一端扫验参与全程通签检流及其闭合分析报送端及后发复联端
