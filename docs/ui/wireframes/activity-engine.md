# 线框图: 泛活动微流建排器 (Activity Builder)

**终端类型**: PC Web (管理后台端)
**相关 Epic**: Epic 05 泛活动生态场景自定制建排器微流引擎

![活动建排器核心线稿](Error:图片需由需求方提供)

## 区域划分

1.  **左侧物料面板 (Component Panel)**
    *   标准表单组件: 单选、多选、下拉、文本、附件上传。
    *   业务域组件: 支付费用块、签到二维码生成器、座位选座画布、电子票据。
2.  **中心画布引擎 (Canvas Workspace)**
    *   可视化流程式排列。
    *   具有对齐网格线。
    *   ![画布拖拽示意图](Error:图片需由需求方提供)
3.  **右侧属性配置器 (Properties Panel)**
    *   当前选中模块的具体属性编辑 (如: 下拉选项内容配置、是否必填、可见性条件过滤)。
4.  **操作底栏 (Action Bar)**
    *   预览微门户生成页 (手机屏幕视图预览器)。
    *   保存草稿 / 正式发布并分配外发渠道链。
