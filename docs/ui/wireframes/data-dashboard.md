## 区域划分

1.  **顶栏 (Header Bar)**
    *   页面标题: “智慧校友服务全域数据驾驶舱”
    *   右侧: 实时时间、刷新控制开关、深/浅色模式切换、全屏按钮。
2.  **左侧栏图表群: 组织与认证态势**
    *   模块 A: 四维网真认证漏斗图 (访客 -> 实名认证 -> 校检比对 -> 发证)。
    *   模块 B: 校友人脉树深层结构拓扑地图 (支持缩放查看某二级分会活跃度)。
3.  **中部主舞台: 实时动态地理沙盘**
    *   全球校友即时签到/活动发散涟漪 3D 地球。
    *   ![地球可视化沙盘](Error:图片需由需求方提供)
4.  **右侧栏图表群: 资金与资源赋能**
    *   模块 C: 数字长碑捐赠滚动明细列表。
    *   模块 D: 大额实项资金募集池进度条/雷达图。
    *   模块 E: 企联池内推简历投递趋势折线图。
